# KAZKIOTI

Next.js 14 + React 18 + TypeScript + Tailwind. Bilingual (RU/KK) marketing site for the Kazakhstan tractor manufacturer.

## Develop

```bash
pnpm install
pnpm dev
```

Open http://localhost:3000 (auto-redirects to `/ru`).

Visual component index: `/ru/_dev` (excluded from production sitemap).

## Build

```bash
pnpm build
pnpm start
```

## Typecheck

```bash
pnpm typecheck
```
